# HOE TE GEBRUIKEN

- Clone de repository (als je dit nog niet hebt gedaan)

- Kijk of er changes zijn, zo ja; pull deze via Github Desktop of de terminal (`git pull`)

- Open het project van de huidige workshop in VS Code (of een andere code editor)

- In de terminal, type `npm i` om de benodigde packages te installeren

- In de terminal, type `npm run dev` om de webapplicatie te starten, en volg vervolgens de link die wordt getoond.
